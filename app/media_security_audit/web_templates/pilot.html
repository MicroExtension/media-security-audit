Setup

System Templates
1
Check appliance status

Confirm authentication, storage, tool visibility, and backup readiness.

2
Select pilot template

Use a restrained audit profile aligned with the approved maintenance scope.

3
Record client context

Keep notes, reference, contacts, and retention details ready before mission setup.

Mission

Dashboard Templates
1
Create client and mission

Use the dashboard form and attach the selected audit template when relevant.

2
Complete authorization

Add authorization reference, contact, dates, emergency contact, and recipients.

3
Approve scope

Mark only validated IP, CIDR, domain, URL, and host entries as approved.

4
Select checks

Keep selected checks aligned with authorization and visible scan plan previews.

Review

Activity Remediations
1
Import or record findings

Use manual findings or guarded CLI outputs that stay inside approved scope.

2
Review dispositions

Add notes for false positives, accepted risks, and counter-test decisions.

3
Validate remediation wording

Use remediation suggestions as a baseline, then adapt to client context.

4
Check activity trace

Confirm mission events show the preparation and review path.

Handoff

Exports Backup
1
Generate reports

Create JSON, Markdown, and HTML reports from the reviewed mission page.

2
Generate mission package

Package reports, readiness, scan plan, manifest, and evidence references.

3
Verify export inventory

Use the export inventory to confirm package integrity before delivery.

4
Create workspace backup

Generate a local backup before closing the pilot session.

Closeout

1
Record residual risks

Keep accepted risks and pending counter-tests visible for the next visit.

2
Save delivery evidence

Keep package manifests, verification files, and export inventory with the mission.

3
Plan next iteration

Use client feedback to tune templates, wording, and remediation priorities.
