Cibles Autorisees

Une cible par ligne

Ajoute uniquement les IP, plages, domaines ou URL presents dans l'autorisation client. Les exclusions seront gerees depuis la mission.

  • Interne CIDR, IP ou nom d'hote pour Nmap, SMB et inventaire LAN.
  • Domaines Domaines publics pour DNS/Mail, TLS et exposition Internet.
  • Web URLs URL HTTPS completes pour les en-tetes HTTP et le TLS Web.
  • AD / LDAP Controleurs de domaine ou hotes LDAP pour l'audit annuaire.

Preparation Du Test Reel

Avant les controles actifs
  • Autorisation conservee L'accord signe peut etre rattache a cette mission.
  • Perimetre verifie Les cibles correspondent au perimetre et aux exclusions.
  • Outils verifies Les outils manquants sont installes ou documentes.
  • Sorties pretes Les attentes PDF, JSON et preuves sont claires.
  • Arret connu Le technicien sait arreter les scans et relancer le service.
  • Fenetre client validee Le creneau et le contact client sont confirmes avant execution.
