# Sprint 77 - Activity Page Shortcuts

## Goal

Make the Activity page easier to navigate by adding direct shortcuts to the
filter controls, visible events, and export links.

## Scope

- Add Activity page shortcut links below the metrics.
- Add stable anchors for filters, events, and exports.
- Show shortcut counts from already-loaded activity view data.
- Update tests and product notes.

## Acceptance Criteria

- Activity pages expose shortcuts for filters, events, and exports.
- Shortcuts target stable in-page anchors.
- Shortcut badges use existing activity view counts.
- Activity filtering and export URLs remain unchanged.

## Safety

- This sprint only changes read-only web navigation.
- No scanner execution is added.
- No network activity is added.
