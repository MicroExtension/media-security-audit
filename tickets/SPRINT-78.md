# Sprint 78 - Remediation Page Shortcuts

## Goal

Make the Remediation Library easier to navigate by adding direct shortcuts to
the filter controls, visible entries, and export links.

## Scope

- Add Remediation Library shortcut links below the metrics.
- Add stable anchors for filters, entries, and exports.
- Show shortcut counts from already-loaded remediation view data.
- Update tests and product notes.

## Acceptance Criteria

- Remediation Library pages expose shortcuts for filters, entries, and exports.
- Shortcuts target stable in-page anchors.
- Shortcut badges use existing remediation view counts.
- Remediation filtering and export URLs remain unchanged.

## Safety

- This sprint only changes read-only web navigation.
- No scanner execution is added.
- No network activity is added.
