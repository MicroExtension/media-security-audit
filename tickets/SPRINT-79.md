# Sprint 79 - Audit Template Page Shortcuts

## Goal

Make the Audit Templates page easier to navigate by adding direct shortcuts to
the filter controls and visible template list.

## Scope

- Add Audit Templates shortcut links below the metrics.
- Add stable anchors for filters and templates.
- Show shortcut counts from already-loaded template view data.
- Update tests and product notes.

## Acceptance Criteria

- Audit Templates pages expose shortcuts for filters and templates.
- Shortcuts target stable in-page anchors.
- Shortcut badges use existing template view counts.
- Template filtering remains unchanged.

## Safety

- This sprint only changes read-only web navigation.
- No scanner execution is added.
- No network activity is added.
