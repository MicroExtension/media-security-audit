# Sprint 80 - System Page Shortcuts

## Goal

Make the System Status page easier to operate by adding direct shortcuts to the
main appliance readiness sections.

## Scope

- Add System Status shortcut links below the metrics.
- Add stable anchors for authentication, storage, inventory, backup, and tools.
- Show shortcut badges from already-loaded system status data.
- Update tests and product notes.

## Acceptance Criteria

- System Status pages expose shortcuts for all main readiness sections.
- Shortcuts target stable in-page anchors.
- Shortcut badges use existing system status values.
- System diagnostics remain read-only.

## Safety

- This sprint only changes read-only web navigation.
- No scanner execution is added.
- No network activity is added.
