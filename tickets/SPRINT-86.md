# Sprint 86 - Anchor Target Context

## Goal

Make in-page shortcuts easier to use by preserving space above anchor targets
and adding a subtle visual context when a section is targeted.

## Scope

- Add shared scroll spacing for anchored elements.
- Add visible target outlines for page sections and export link groups.
- Update tests and product notes.

## Acceptance Criteria

- In-page shortcut targets have consistent top scroll spacing.
- Targeted sections and export groups show a restrained outline.
- Existing shortcuts, routes, data, and forms remain unchanged.
- Tests cover the shared anchor target styling.

## Safety

- This sprint only changes shared web styling.
- No scanner execution is added.
- No network activity is added.
